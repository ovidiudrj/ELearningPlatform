# 📚 E-Learning Platform (Work in Progress)

A Full-Stack educational platform built to facilitate access to courses and learning materials.

## 🛠️ Tech Stack
- **Frontend:** Angular, TypeScript, HTML/CSS, Bootstrap
- **Backend:** C#, ASP.NET Core REST API
- **Database:** SQL Server & Entity Framework Core

## 🚀 Current Status
So far, the core infrastructure has been successfully established:
- Connected the SQL Server database via Entity Framework.
- Created .NET Core API endpoints for data retrieval.
- Configured CORS policies and API routing.
- Built a responsive Single Page Application (SPA) using Angular and Bootstrap, capable of asynchronously fetching and dynamically displaying real data from the backend.
- 
The long-term vision is to evolve this project into a comprehensive educational portal.
